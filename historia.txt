pruebas del programa git bash

git add para agregar las modificaciones al repositorio
git commit -m "   ": para aumentar mensaje de programacion
git log: para mostrar los cambios
git status: comprobar como esta el documento
git diff: para ver los cambios del archivo
git checkout: para saltar de ramas del proyecto que se tiene
git branch: agregar una rama al proyecto para que trabajen
